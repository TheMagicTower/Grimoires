Codex는 코드 생성 전문 Familiar입니다.

## When to Use

- 새 코드 작성이 필요할 때
- 기존 코드 수정/리팩토링
- 버그 수정
- 테스트 코드 작성

## How to Invoke

MCP 서버가 있으면 Codex 도구를 사용하고, 없으면 Bash로:
```bash
codex exec "요청 내용"
```
