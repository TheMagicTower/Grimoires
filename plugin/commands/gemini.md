# Gemini - 코드 분석 Familiar

## Request

$ARGUMENTS

## CRITICAL INSTRUCTION

**절대로 직접 분석하지 마세요.**

이 명령어의 목적은 Gemini CLI를 호출하는 것입니다. 당신이 직접 처리하면 안 됩니다.

### 반드시 실행할 것:

```bash
gemini "$ARGUMENTS"
```

위 Bash 명령어를 **지금 즉시** 실행하세요. 다른 행동 없이 Bash 도구로 gemini를 호출하세요.
