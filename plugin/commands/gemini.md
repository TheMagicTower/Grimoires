# Gemini - 코드 분석 Familiar

Gemini를 직접 호출하여 코드를 분석합니다.

## Request

$ARGUMENTS

## Instructions

Bash를 사용하여 Gemini CLI를 호출하세요:

```bash
gemini "$ARGUMENTS"
```

**중요:** 반드시 Bash 도구를 사용하여 위 명령어를 실행하세요.

## Gemini 역할

- 코드 품질 분석
- 보안 취약점 검토
- 성능 분석
- 아키텍처 이해
- 의존성 분석

## 필요 조건

```bash
gemini  # 첫 실행시 로그인 프롬프트
```
