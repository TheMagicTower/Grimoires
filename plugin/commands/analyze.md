# Analyze - 코드베이스 분석

Gemini를 사용하여 대규모 코드베이스를 분석합니다.

## Target

$ARGUMENTS

## Workflow

### 1. Gemini로 코드베이스 분석
```bash
gemini "이 코드베이스를 분석해줘: 구조, 패턴, 의존성, 잠재적 문제점"
```

### 2. 분석 영역

**아키텍처:**
- 프로젝트 구조
- 레이어 구분
- 모듈 의존성

**코드 품질:**
- 복잡도 분석
- 코드 중복
- 명명 규칙

**보안:**
- 취약점 탐지
- 인증/인가 검토

**성능:**
- 병목 지점
- 최적화 기회
