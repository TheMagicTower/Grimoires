# Sprint Review

Batch review sprint PRs against completion criteria, coding conventions, scope boundaries, and edge cases.

## Task

$ARGUMENTS

## Usage

```
/sprint:review              # Review all open PRs
/sprint:review 143 144 145  # Review specific PRs
```

## Review Checklist

- Completion criteria vs sprint plan
- Coding conventions per squad
- Scope boundary enforcement
- Edge case coverage
- Custom project checks (from config)
